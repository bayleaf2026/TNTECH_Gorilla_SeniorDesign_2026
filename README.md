# TNTECH_Gorilla_SeniorDesign_2026
This project aims to design and construct a bioinspired quadruped robot that models gorilla movement, with a focus on knuckle-walking biomechanics. Developed in collaboration with the Tennessee Technological University Paleontology Department, the system serves as a platform for studying anatomical motion and load distribution.
